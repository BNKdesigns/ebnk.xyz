## ‚úÖ Requirements

Before getting started, ensure you have the following:

- A **Samsung Tizen Smart TV** with **Developer Mode enabled** ([Click here to learn how to turn it on.](https://developer.samsung.com/smarttv/develop/getting-started/using-sdk/tv-device.html#Connecting-the-TV-and-SDK) Make sure to NOT follow step 2/Tizen Studio part) 
- A **Default webbrowser**  
- A valid **[Samsung Account](https://account.samsung.com/iam/signup)** (required for Tizen 7+)
